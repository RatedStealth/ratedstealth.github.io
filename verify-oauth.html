Troubleshooting

If authorization always redirects to Kick homepage:

  • The most common issue is a redirect URI mismatch. Even a trailing slash can break it.
  • Make sure you've created an API application in Kick Developer Portal
  • Verify that your session with Kick is active (try logging in again)
  • Check that your application is approved and active in Kick
